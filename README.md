# Scan-local-networks
## Objective 
  Learn to discover open ports on devices in your local network to understand network exposure. 
## Tools Used 
  - Nmap - Wireshark (optional)
## Steps Performed 
  1. Installed Nmap from the official website.
  2. Identified local IP range (e.g., 192.168.1.0/24).

  4. Recorded IP addresses and open ports.
  5. Optionally analyzed packet capture with Wireshark.

  7. Identified potential security risks.
  8. Saved scan results in text format.

## Outcome
- Gained basic network reconnaissance skills.
- Understood network service exposure.
